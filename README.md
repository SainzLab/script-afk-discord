# script-afk-discord

Token User
1. Buka window Developer Tools (F10).
2. Di halaman Discord (di balik layar hitam itu), ketik pesan chat apa saja (misalnya "p" atau "tes") ke teman atau ke server mana saja, lalu Kirim/Enter.
3. Perhatikan kolom di bawah tulisan "Name". Setelah kamu tekan Enter, akan muncul tulisan baru bernama messages di situ.

Setelah tulisan messages muncul:
1. Klik kiri tulisan messages tersebut.
2. Akan muncul panel baru di sebelah kanan.
3. Pilih tab Headers.
4. Scroll ke bawah cari bagian Request Headers.
5. Cari tulisan authorization. Kode acak di sebelah kanannya adalah TOKEN anda.

Untuk VC_ID (ID Voice Channel):
1. Di aplikasi Discord, cari Voice Channel yang mau dimasuki.
2. Klik Kanan pada nama channel tersebut.
3. Pilih Copy Channel ID (paling bawah). Kalau menu ini tidak ada, nyalakan dulu Developer Mode di User Settings > Advanced.
